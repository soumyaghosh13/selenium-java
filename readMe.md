# HRAcuity

Clone this project from Bitbucket.

## Installation



```bash
install java 8
set Environment Variable 
set JAVA_HOME
```
* Install any IDE ( Eclipse, Intellij )
* Install Needed Plugins (Cucumber, TestNG )
* Load Project in IDE
* Run Testrunner with Preferred Tag.
